### info.txt 

Contains the info on posts.

variables:
X1: id
X2: url
X3: name
X4: date
X5: tags
X6: blog url
X7: blog name
X8: votes


### blog_info.txt 

Contains the info on blogs.

variables:
X1: blog url
X2: blog name
X3: blog description
X4: creation date
X5: type
X6: posts
X7: sport
X8: tags
X9: author's profile url


### post_text.zip 

Containes the text of all posts listed in info.txt and a few more. Filenames contain post ids.
